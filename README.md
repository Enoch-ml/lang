# translation practice
This is a tool that allows for translation practice from English into the CJK languages.
Dictionaries are `.tsv` found in `/dicts/`, and adding a new dict requires updating the `pageConfig` variable in `vocab.html`.
There are various basic vocab supported from the CJK languages.

There are supported functionalities for two sets of workbooks:
* Cheng & Tsui: Integrated Chinese 中文听说读写 1-4
* Genki: An Integrated Course in Elementary Japanese

There are also flashcards for words "Important for Travel".

Additionally, this page has supported functionality for uploading your own flashcards as a tsv with headers (`pages/flashcard.html`).


This repository is open to access at [enoch-ml.github.io/lang/](https://enoch-ml.github.io/lang/).
